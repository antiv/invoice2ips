# Antigravity Instructions

This file contains context and rules for the Antigravity AI assistant while working on the `invoice2ips` project.

## Project Stack

- **Frontend**: React, Vite, TailwindCSS
- **Desktop/Mobile**: Tauri (Rust)
- **Language**: TypeScript throughout

## Common Commands

- Web Development: `npm run dev`
- Tauri Development: `npm run tauri:dev`
- Tauri Build: `npm run tauri:build`
